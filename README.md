# Simple Calculator (WinForms)

A simple calculator made with **C# and Windows Forms**.  
It is a full rewrite of an **[earlier version built with Python and Tkinter](https://github.com/Moamen-Tamer/tkinter-simple-calculator)** ‚Äî same colors, same layout, same features, different language.

---

## Features
- Basic arithmetic: +, -, √ó, √∑ (one operation at a time)
- Clear (AC), backspace (‚Üµ), and percentage (%) buttons
- Dark/Light mode toggle with icons
- Realistic calculator layout
- Division by zero protection

---

## Colors

**Dark Mode**
- Background: `#1e1e1e`
- Buttons: `#333333`
- Numbers: `#f5f5f5`
- Clear/Delete: `#ff1a1a`
- Result: `#ffe4c4`

**Light Mode**
- Background: `#f5f5f5`
- Buttons: `#dedeff`
- Numbers: `#000000`
- Clear/Delete: `#ff0000`
- Result: `#4d2600`

---

## Requirements
- Windows
- .NET SDK (6.0 or later)
- `light.png` and `dark.png` icon files placed at `D:\VS\WORKSHOP\`
